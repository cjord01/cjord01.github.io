Instance Variables vs. Class Variables

You've probably noticed how there are times when you see variables with an '@' at the beginning and other times you see a variable with '@@' at the beginning. What's up with that?

Variables that start with one @ sign like this — '@variable' — are called Instance Variables.

Variables that start with two @ signs like this — '@@variable' — are called Class Variables.

Both these variables are used while defining a method within a class. The difference is that Instance Variables are specific to each individual instance of a class, and Class Variables are specific to the class itself. That means that if you change the Class Variable then it affects all the instances of that class.
