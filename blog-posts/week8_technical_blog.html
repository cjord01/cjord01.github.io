What is Rails?

I'm now eight weeks into prepping for Dev Bootcamp. I've spend many hours learning the Ruby language but if you were to ask me what the 'Rails' part of 'Ruby on Rails' is, I wouldn't have much of an answer for you. It's time to figure it out.

'Ruby' and 'Ruby on Rails' are different. Ruby is just the programming language. Ruby on Rails (or just Rails) is the web application framework which runs the Ruby language. Let's go a little deeper.

Here are some of the characteristics of Rails:

  • It's open source.
  • It's a full-stack framework which means that it's able to gather multiple libraries to form a single cohesive software stack.
  • It's capable of gathering information from the server, and it can interact and retrieve information from a database.
  • It features a routing system that is independent of the web server.
  • Using convention over configuration, Rails makes building web apps simpler. This generally means less code and less repetition.
