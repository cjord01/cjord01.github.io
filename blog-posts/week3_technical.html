Week #3: Technical

JavaScript is a programming language used to create interactive effects.

Why is JavaScript popular in web development?

JavaScript is "the best language for the web." It fully supports web development and is very responsive and interactive in creating HTML files. Developers with strong JavaScript skills are in high demand. Companies big and small are desperately looking for Front End developers and UI developers.

What are the great things about JavaScript?

  • JS is relatively easy to learn.
  • All the major browsers support JS.
  • JS works well with mobile devices.
  • JS has many libraries so it's easy to get a site built quickly.
  • JS is fast and it reduces the load on the server.

What are it's limitations?

  • JS cannot directly access files on the user's system.
  • JS cannot access databases.
  • JS has some security issues.
  • Rendering in JS can be inconsistent.
  • JS does not have multithreading.
